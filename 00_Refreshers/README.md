# 00_Refreshers · 基础盲点补课箱

按需使用。学习主线时遇到某个**基础概念不够熟**,就在这里开一个小笔记,不打断主进度。

## 适合放进来的内容

- 某个优化器的推导(AdamW weight decay 的正确实现)
- Normalization 对比(LayerNorm / RMSNorm / 数值稳定性)
- Softmax / log-softmax / cross-entropy 的数值技巧
- 反向传播里某个具体算子的梯度推导
- PyTorch 某个 API 踩坑记录
- 矩阵维度/广播规则复习

## 命名建议

`topic_slug.md`,例如:
- `rmsnorm_vs_layernorm.md`
- `adamw_weight_decay.md`
- `logsumexp_trick.md`
